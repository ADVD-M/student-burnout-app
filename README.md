## 📋 Overview

This application uses a **Random Forest classification model** trained on real student mental health survey data to predict burnout risk as **Low**, **Medium**, or **High**. It features:

- 🎯 **Risk Prediction Form** — fill in academic and lifestyle details for an instant assessment
- 📊 **Results Dashboard** — confusion matrix, feature importance, accuracy, F1, ROC AUC
- 🤖 **AI Wellness Chatbot** — evidence-based breathing, journaling, focus, and stress exercises

> ⚠️ **Disclaimer**: This tool is for informational and educational purposes only. It is not a substitute for professional mental health advice or treatment.

## 🚀 Quick Start

### 1. Clone / Download the project

```bash
git clone https://github.com/yourusername/student_burnout_app.git
cd student_burnout_app
```

### 2. Set up a virtual environment

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Place the dataset

Copy `mentalhealth_dataset.csv` into the `data/` folder:

```
data/mentalhealth_dataset.csv
```

> **Shortcut**: If `mentalhealth_dataset.csv` is on your Desktop, the `train.py` script will automatically copy it to `data/`.

### 5. Train the model

```bash
python train.py
```

This will:
- Load and preprocess the dataset
- Engineer the `BurnoutRisk` composite label (Low / Medium / High)
- Train a Random Forest classifier
- Print evaluation metrics (accuracy, F1, ROC AUC, confusion matrix)
- Save the model to `models/burnout_model.pkl`
- Save metrics to `models/metrics.json`
- Run a smoke-test prediction

### 6. Launch the app

```bash
streamlit run app/Home.py
```

Open your browser at `http://localhost:8501`.

## 🌐 Deploy to Streamlit Cloud

1. Push this repository to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io) → **New app**
3. Set:
   - **Repository**: `yourusername/student_burnout_app`
   - **Branch**: `main`
   - **Main file path**: `app/Home.py`
4. Add `mentalhealth_dataset.csv` to the `data/` folder and commit it (remove `data/*.csv` from `.gitignore` for Cloud)
5. Deploy — Streamlit Cloud will auto-install `requirements.txt`

> **Note**: You need to run `train.py` locally and commit `models/burnout_model.pkl` and `models/metrics.json` to the repo (remove their gitignore rules), OR implement a training step in the app itself.

## 🧠 Dataset

**Source**: Student Mental Health Survey (1001 rows, 15 features)

| Column | Type | Description |
|--------|------|-------------|
| Gender | Categorical | Male / Female |
| Age | Numerical | Student age |
| Course | Categorical | Academic major |
| YearOfStudy | Categorical | Year 1–4 |
| CGPA | Numerical | GPA (0–4.0) |
| Depression | Binary | 0/1 |
| Anxiety | Binary | 0/1 |
| PanicAttack | Binary | 0/1 |
| SpecialistTreatment | Binary | 0/1 |
| SymptomFrequency_Last7Days | Numerical | 0–7 days |
| HasMentalHealthSupport | Binary | 0/1 |
| SleepQuality | Numerical | 1–5 (higher = better) |
| StudyStressLevel | Numerical | 1–5 (higher = worse) |
| StudyHoursPerWeek | Numerical | 1–40 |
| AcademicEngagement | Numerical | 1–5 |

**Burnout Label Engineering** (composite score → category):
- `Depression` + `Anxiety` + `PanicAttack` + low sleep + high stress + high symptom frequency → score 0–12
- Score ≤ 3 → **Low** | 4–7 → **Medium** | 8–12 → **High**
